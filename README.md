# Jogo do número Secreto

Este projeto consiste em um programa de chutes do usuário para acertar um numero aleatório entre 1 e 10.

### Tecnologias Utilizadas

- **JavaScript:**
  - A linguagem de programação.
- **HTML**
  - Linguagem de Marcação.
- **CSS**
  - Estilização do site.
